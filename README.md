# 基于Java的零食超市购物系统<br>

（源代码+12000字文档+ppt）

## 项目简介

基于Java的零食超市购物系统，是一套包含了管理员上架商品、用户购物、购物车、支付等一系列操作的购物系统。<br>
以Spring框架为基础，数据库使用的是mysql数据库。<br>
该系统功能完善，界面美观，非常适合作为毕设或者课程作业以及新手开发学习。<br>
